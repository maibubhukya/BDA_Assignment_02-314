## 🧪 Experiments Implemented

### 🔹 1. Classification Model using Spark
**Dataset:** Heart Disease Dataset  
**Algorithm:** Logistic Regression  

**Objective:**  
Predict whether a person has heart disease based on medical attributes.

**Steps Performed**
- Data loading using Spark DataFrame
- Feature Vector creation using VectorAssembler
- Train-Test split (80-20)
- Model training using Logistic Regression
- Model evaluation using Accuracy

---

### 🔹 2. Clustering Model using Spark
**Dataset:** Iris Dataset  
**Algorithm:** K-Means Clustering  

**Objective:**  
Group flowers into clusters based on petal and sepal measurements.

**Steps Performed**
- Data preprocessing
- Feature vector creation
- KMeans model training
- Cluster prediction
- Visualization of clusters

---

### 🔹 3. Recommendation Engine using Spark
**Dataset:** MovieLens Dataset  
**Algorithm:** ALS (Collaborative Filtering)

**Objective:**  
Recommend movies to users based on their previous ratings.

**Steps Performed**
- Load ratings dataset
- Train ALS model
- Predict ratings
- Evaluate using RMSE
- Generate top movie recommendations
